# Logo Files

Place your logo PNG files here:

- `georgia-tech.png` - Georgia Tech logo
- `palantir.png` - Palantir logo
- `blackstone.png` - Blackstone logo
- `uw-madison.png` - UW Madison logo

**Requirements:**

- Format: PNG with transparency (opaque backgrounds are fine)
- Recommended size: 200px width (height will be auto-scaled)
- Logos will be displayed in white (inverted) so any color logo will work

The carousel will automatically rotate these logos.
